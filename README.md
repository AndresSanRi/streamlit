Dashboard de Producción Industrial con Streamlit

Este proyecto presenta una plataforma interactiva de Business Intelligence (BI) desarrollada en Python, diseñada para transformar datos operativos en indicadores clave de rendimiento (KPIs) para el sector de manufactura.

A través de una adaptación técnica de un dataset de ventas, la aplicación simula el monitoreo de líneas de producción, eficiencia operativa y control de calidad.

Funcionalidades Principales
Monitoreo en Tiempo Real de KPIs: Visualización instantánea de Producción Total, Tasa de Defectos, Tiempos de Operación y Productividad por Minuto.
Filtros Dinámicos: Segmentación de datos por Línea de Producción y Turno Operativo mediante una barra lateral interactiva.
Análisis Temporal: Gráficos de líneas interactivos que permiten identificar tendencias y picos de producción diaria.
Control de Calidad: Visualización de mapas de calor por líneas de producción para identificar cuellos de botella o fallos recurrentes.
Gestión de Datos: Acceso directo a la tabla de registros filtrados para auditoría de datos.📊

Indicadores Implementados (KPIs)
El dashboard calcula automáticamente las siguientes métricas industriales:
Producción Total: Sumatoria de unidades procesadas.
Tasa de Defectos: Porcentaje de unidades no conformes sobre el total producido ($Defectos / Producción$).
Productividad por Minuto: Ratio de eficiencia de salida frente al tiempo de operación programado.
